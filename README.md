# Todo List In Flutter

Todo list

## Feature
- Add todo
- Display todo
- Check todo
- Sort todo by category
- Dark mode
- Drawer
- Dialog at succes add todo
- Count todo uncheck in drawer
- Display todo sort by check and uncheck
